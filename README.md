# Что тут есть ?
Уникальное решение, помогает юристам экономить кучу времени.
Мы парсим данные в базу данных. Юристы используя веб-доступ получают эти данные !
------------------------------------------------------
# Из чего состоит решение ?
1.  Frontend - визуальная часть решения
1.1 Получение данных из БД используя фильтры
1.2 Создание очереди задач для парсинга новых данных

Используемые технологии: vue.js

2.  Backend
2.1 Парсинг данных порталов используя Get и Post запросы
2.2 Web.Api для frontend-части. Получает запросы на получение кейсов, извлекает кейсы из БД Postgres, отдает кейсы пользователям.
2.2 Десктоп решение на WPF сделано специально для портала ГАС Правосудие на Selenium

Используемые технологии: ASP.Net Core 3, WPF, PostgreSql
